# Потоки данных

| Поток | Источник | Получатель | Защита | Примечание |
|---|---|---|---|---|
| Работа пользователя | Браузер | Caddy | HTTPS, VPN или внутренняя сеть | На сервер публикуется только `443` |
| Авторизация | Браузер | IAM API | JWT access token, HttpOnly refresh cookie | Cookie в production только `Secure` |
| Юридические данные | API | PostgreSQL | Закрытая Docker-сеть | Порт БД наружу не публикуется |
| События | API и workers | Redis | Закрытая Docker-сеть | Включены AOF `everysec` и RDB snapshots |
| Платежи 1С | Интеграционный шлюз | Internal API | Отдельная приватная точка входа, `X-Internal-Key` | Публичный nginx возвращает `404` |
| Метрики | Prometheus | API и exporters | Только внутренняя сеть | `/metrics` нельзя публиковать наружу |
| Backup | PostgreSQL | Внешнее хранилище | Шифрование и ограниченный доступ | Место хранения согласовать |

## Категории данных

- Учётные записи сотрудников: ФИО, email, роль, филиал.
- Юридические дела и претензии: контрагенты, БИН, суммы, процессуальные события.
- Аудит действий и технические логи.
- Файлы документов: функциональность хранения содержимого требует отдельного согласования.
