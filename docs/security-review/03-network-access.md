# Сетевой доступ

## Разрешённые подключения к серверу

| Порт | Протокол | Источник | Назначение | Статус |
|---|---|---|---|---|
| `443` | TCP | Корпоративная сеть или VPN | HTTPS приложения | Разрешить |
| `80` | TCP | Корпоративная сеть или VPN | Redirect на HTTPS и ACME при необходимости | Ограничить |
| `22` | TCP | Административная подсеть | SSH по ключам | Ограничить allowlist |

## Запрещённая публикация наружу

PostgreSQL `5432`, Redis `6379`, API-порты `8001-8004`, Prometheus `9090`, Grafana `3000`,
Alertmanager `9093` и Docker socket не публикуются во внешнюю сеть.

`/api/internal/*` заблокирован на публичном nginx. Для 1С требуется отдельная приватная точка
входа с allowlist адресов, журналированием и ротацией `INTERNAL_API_KEY`.
