# Известные риски

| Риск | Приоритет | Статус | Условие закрытия |
|---|---|---|---|
| Общие demo-пароли могли появляться при пустой БД | Критический | Исправлено в коде | Проверить отсутствие demo-аккаунтов в целевой БД |
| Бухгалтер мог изменять претензии и дедлайны | Критический | Исправлено и проверено | Повторять интеграционный RBAC-тест в CI |
| Филиальный юрист мог указать чужой филиал в претензии | Критический | Исправлено и проверено | Повторять интеграционный RBAC-тест в CI |
| `/health` мог вернуть HTML SPA | Высокий | Исправлено и проверено локально | Проверить `/readyz` после деплоя |
| Internal API был доступен через публичный nginx | Высокий | Исправлено на edge | Спроектировать приватный доступ 1С |
| Единственный физический сервер | Высокий | Принято для пилота | Согласовать допустимый простой и внешний backup |
| Backup пока не копируется за пределы сервера | Высокий | До пилота | Выбрать место хранения и выполнить restore drill |
| Нет пагинации списка дел | Средний | До пилота | Добавить пагинацию и провести нагрузочный тест |
| Мониторинг и алерты не развёрнуты | Средний | До пилота | Поднять Prometheus, Grafana и Alertmanager |
| PgBouncer не подключён | Средний | До масштабирования | Настроить после замера числа DB connections |
| Frontend dependency audit содержал замечания | Средний | Исправлено | Повторять `npm audit --omit=dev` в CI |
| Загрузка содержимого документов не реализована | Средний | Ограничение функции | Не заявлять функцию до отдельного проектирования |
