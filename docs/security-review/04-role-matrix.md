# Матрица ролей

| Возможность | Директор | Главный юрист | Филиальный юрист | Бухгалтер |
|---|---|---|---|---|
| Просмотр всех филиалов | Да | Да | Только центральный аппарат | Да |
| Просмотр своего филиала | Да | Да | Да | Да |
| Создание и изменение дел | Да | Да | Только свой филиал | Нет |
| Создание и изменение претензий | Да | Да | Только свой филиал | Нет |
| Изменение процессуальных дедлайнов | Да | Да | Только свой филиал | Нет |
| Просмотр отчётов | Да | Да | В пределах доступных данных | Да |
| Управление пользователями | Уточнить регламент | Уточнить регламент | Нет | Нет |

## Правила

- Роль `accountant` является read-only.
- Филиальный юрист не может создать или перенести претензию в другой филиал.
- Привязанное к претензии дело и назначенный филиальный юрист должны относиться к тому же филиалу.
- Матрицу необходимо утвердить с владельцем процесса до промышленного запуска.
