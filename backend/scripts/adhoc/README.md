# Adhoc scripts (не поддерживается)

Одноразовые исследовательские снайпеты, оставшиеся от импортов xlsx и дебага
конкретных кейсов (`find_syrlybaev*.py`, `check_2026_counts.py`, и т.п.).

- Многие из них хардкодят пути к xlsx, которых уже **нет в репо** (PII вынесена).
- Часть импортирует устаревшие модули (`app.models.case`, `app.db.session.async_session`)
  и **не запустится** без правок.
- Эти скрипты не покрыты тестами и не используются продакшен-кодом или CI.

Если что-то отсюда нужно для воспроизводимой задачи — перепишите как именованный
скрипт в `backend/scripts/` (по аналогии с `import_pir_demo_from_xlsx.py`).
Иначе кандидаты на удаление.
